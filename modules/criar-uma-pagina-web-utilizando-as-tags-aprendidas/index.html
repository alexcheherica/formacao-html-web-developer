<small>: Texto menor.

<u>: Texto sublinhado.

Tags para listas

<ol>: Lista ordenada(numerada).

Exemplo:

  1. Item 1
  2. Item 2

<ul>: Lista não ordenada(com bolinhas).

Exemplo:

  • Item A
  • Item B

<li>: Item da lista (usado dentro de <ol> ou <ul>).
